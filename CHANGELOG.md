# Changelog

## [2.0.0] - 2026-06-23

### Sprint learnings

* Uso de GitHub Flow para trabajo colaborativo.
* Creación y administración de ramas feature.
* Uso de Pull Requests para integrar cambios.
* Resolución manual de conflictos de merge.
* Trabajo coordinado mediante revisión de código.

### Added

* Estructura HTML semántica con `header`, `nav`, `main`, `section` y `footer`.
* Sección de portafolio.
* Formulario accesible con etiquetas `label`.
* Diseño responsivo mediante CSS.
* Variables CSS y transiciones.
* Validaciones de formulario con JavaScript.
* Persistencia de datos utilizando `localStorage`.
* Archivo `README.md`.
* Archivo `CHANGELOG.md`.

### Changed

* Reorganización del contenido de `index.html` para utilizar etiquetas semánticas.
* Mejora de la estructura visual de la página.
* Integración de cambios provenientes de distintas ramas de desarrollo.

### Fixed

* Resolución manual de conflicto de merge en `index.html`.
* Resolución manual de conflicto de merge en `js/app.js`.
* Corrección de inconsistencias durante la integración de cambios mediante Pull Requests.

### Removed

* No se eliminaron funcionalidades durante este sprint.
